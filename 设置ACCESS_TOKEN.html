设置ACCESS_TOKEN

  • 点击右上角你的头像

  • 点击下方的Settings按钮

  • 点击左下方的Developer settings按钮

  • 点击左侧的Personal access tokens按钮

  • 点击Tokens (classic)按钮

  • 点击右上方的Generate new token按钮

  • 点击Generate new token (classic)按钮

  • Note里填用途, 用于提醒你这是干什么的, 你可以直接填项目名称, 比如XXX-Wiki

  • 将其设置为永不过期

  • 勾选repo下的所有选项

  • 点击最下方的Generate token按钮

  • 复制生成的token(关闭页面后你就再也无法查看它了)

  • 回到XXX-Wiki仓库页面

  • 点击右上角的Settings按钮

  • 点击左下方的Secrets and variables按钮

  • 点击Actions按钮

  • 点击右上角绿色的New repository secret按钮

  • Name填写ACCESS_TOKEN, Secret填写你刚才复制的东西

  • 点击下方绿色的Add secret按钮
